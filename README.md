## 运行方法：
要使用可视化界面，运行main.py即可
点击“导入图片”进入选择图片的界面后，进入image文件夹，可随意选择文件夹中的图片进行导入。

## 训练方法：
train.py是训练代码，需要单独运行。如果没有CUDA的话建议使用google云硬盘的GPU来跑
